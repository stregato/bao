# Introduction

PBao is a Python binding for the Bao library. It enables encrypted data storage and exchange. Similar to traditional systems, Bao offers controlled access with a security model inspired by Unix permissions.
Unlike traditional storage, the access is based on cryptographic features, enabling distributed control.

For more details, please refer to [http://github.com/stregato/bao](http://github.com/stregato/bao)
